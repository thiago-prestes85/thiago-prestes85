Glossary

HTML
HyperText Markup Language, used to structure web pages.
CSS
Cascading Style Sheets, used for styling web pages.
JavaScript
A programming language that makes web pages interactive.
URL
Uniform Resource Locator, the address of a web page.
DOM
Document Object Model, a structured representation of HTML elements in a webpage.
